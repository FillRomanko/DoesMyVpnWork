<div align="center">

**🚀 Does my VPN work?** — Кроссплатформенное PWA-приложение для диагностики интернет-соединения, доступное в браузере и устанавливаемое как приложение на любое устройство. Работает даже офлайн, определяя текущее состояние сети — отсутствие подключения, ограниченный доступ («белый список»), нормальный интернет или доступ к заблокированным в РФ ресурсам.
</div>

## ✨ Функции

- 🌐 **Диагностика сети**: определение состояния подключения (офлайн, ограниченный доступ, полный интернет, доступ к заблокированным ресурсам)
- ⚙️ **Фоновая проверка** соединения через Service Workers — работает даже без активного интернета
- 📉 **Аналитика доступности**: успешность запросов видна в релаьном времени
- 🧭 **Тестирование обхода ограничений**: анализ возможности доступа к различным типам ресурсов
- 📲 **PWA-функциональность**: установка и работа на любом устройстве — смартфоне, планшете или ПК
- 📱 **Адаптивный UI** для смартфонов и ПК
- 🎉 **Простая и быстрая установка**: достаточно один раз зайти на сайт или можно установить за минуту нажав на "Установить на моё устройство"
